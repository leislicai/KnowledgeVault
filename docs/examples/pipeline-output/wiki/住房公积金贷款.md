# 住房公积金个人住房贷款

## 政策概述

住房公积金个人住房贷款（以下简称"公积金贷款"）是指住房公积金管理中心运用职工缴存的住房公积金，向符合条件的缴存人发放的用于购买、建造、翻建、大修自住住房的专项低息贷款。其核心优势在于利率显著低于同期商业银行个人住房贷款，能够有效降低职工购房利息负担。

以天水市为例，2021年全市0.33万户家庭享受了住房公积金贷款优惠政策，相较商业贷款基准利率，共减少职工购房利息支出1103.76万元。同年，天水市发放住房公积金个人贷款0.33万笔、13.11亿元，累计发放个人住房贷款3.78万笔、115.62亿元，累计回收贷款39.4亿元。年末个贷率为93.34%，个人住房贷款市场占有率为22.23%，住房贡献率（个人住房贷款发放额与住房消费提取额的总和与当年缴存额的比率）达106.47%。

### 还款方式

贷款回收以"月冲还贷"为主要方式，即委托按月从缴存人住房公积金账户中划扣资金偿还贷款本息。2021年天水市月冲还贷金额占回收本金的65.11%。

## 适用条件

### 基本申请条件

缴存人在申请住房公积金贷款时，需满足以下基本条件：

1. **连续足额缴存**：申请贷款前需连续、按时、足额缴存住房公积金达到规定期限（账户启封后需重新连续足额缴存六个月方能申请）。
2. **未履行按时足额缴存义务的，不得申请住房公积金贷款。**
3. 所购房屋需符合住房公积金贷款规定的范围（新房、二手房、自住住房建造/翻建/大修）。
4. 有个人住房公积金贷款未结清的，不能办理销户业务。

### 特殊人群规定（农业转移人员）

针对农业转移人员等灵活就业群体，除基本条件外还须满足：

- 公积金贷款月还款额不得超过月缴存额。
- 公积金贷款担保方式为所购房屋以外的本地现房作为抵押。
- 账户因欠缴三个月以上被封存的，启封后需重新连续足额缴存六个月方能申请贷款。

### 不予贷款的情形

- 以提供虚假材料等手段骗取住房公积金贷款的。
- 未按规定履行正常缴存住房公积金义务的。

## 办理材料

根据现有资料，办理住房公积金贷款需准备以下材料（具体以当地公积金管理中心要求为准）：

1. **身份证明**：借款申请人及配偶的有效身份证件、户口簿或婚姻证明。
2. **缴存证明**：住房公积金缴存证明（反映连续足额缴存记录）。
3. **购房证明**：
   - 购买新房的：购房合同、首付款发票或收据。
   - 购买二手房的：购房合同、过户证明材料。
   - 建造、翻建、大修自住住房的：相关审批文件及工程预算。
4. **收入证明**：收入证明或银行流水（用于评估还款能力）。
5. **担保材料**：所购房屋以外的本地现房作为抵押的相关权属证明（农业转移人员适用）。

> 注：具体材料清单以当地住房公积金管理中心最新要求为准。

## 办理流程

### 一般流程

1. **提出申请**：缴存人向当地住房公积金管理中心或受委托银行提交贷款申请及相关材料。
2. **中心审核**：公积金管理中心对申请人资格、缴存情况、还款能力、购房真实性等进行审核。
3. **签订合同**：审核通过后，申请人与管理中心签订借款合同，并办理抵押登记等担保手续。
4. **发放贷款**：管理中心委托银行将贷款资金划转至售房单位或借款人指定账户。
5. **按期还款**：借款人按合同约定方式（通常为月冲还贷或委托扣款）偿还贷款本息。

### 违约处理流程

借款人取得公积金贷款后不按规定履行正常缴存义务的，即构成违约，管理中心有权：

1. 解除借款合同并提前收回贷款本息；
2. 将借款人信用情况记入人民银行征信系统；
3. 记入黑名单并依法向社会公告；
4. 拒不偿还贷款的，依法向人民法院提起诉讼。

### 骗取贷款的处罚

以提供虚假材料等手段获得公积金贷款的：

1. 由管理中心收回所骗取的全部贷款；
2. 将责任人纳入人民银行征信系统并记入黑名单，依法向社会公告；
3. 取消缴存人5年内住房公积金提取及贷款资格；
4. 构成犯罪的，依法追究刑事责任。

## 相关依据

1. **《住房公积金管理条例》**——住房公积金制度的基本法规，各地政策可在其框架内适时调整。
2. **《天水市住房公积金流动性风险预警管理办法》**——2021年经天水市住房公积金四届五次管委会审议通过，规范贷款发放，保障资金流动性。
3. **《天水市住房公积金管理中心关于农业转移人员缴存和使用住房公积金管理办法》**——2020年11月14日发布施行，对农业转移人员的缴存、贷款申请、担保方式、违约处理等作出专项规定。
4. **《天水市住房公积金管理中心2021年度报告》**——含年度贷款发放总量、结构分析、利息减免成效等核心数据。
