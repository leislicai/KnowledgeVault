# 关于开展治理违规提取住房公积金工作的通知（建金〔2018〕46号）

## 政策概述

《关于开展治理违规提取住房公积金工作的通知》（建金〔2018〕46号）是由住房城乡建设部、财政部、人民银行、公安部于2018年联合发布的规范性文件。该通知针对近年来一些机构和个人通过伪造证明材料、虚构住房消费行为等手段违规提取住房公积金，甚至形成骗提套取住房公积金"黑色产业链"的突出问题，提出了系统治理措施。——来源：`kb_325`

通知以"房子是用来住的，不是用来炒的"为根本定位，围绕规范改进提取政策、优化审核流程、实施失信联合惩戒、加强内部风险管理、推进部门信息共享、建立跨地协查机制、集中开展治理工作、广泛开展宣传引导等八个方面作出全面部署，旨在保证住房公积金制度稳健运行，依法维护缴存职工权益。——来源：`kb_325`, `kb_326`

通知的适用范围覆盖全国各省、自治区、直辖市住房城乡建设、财政、公安系统以及人民银行各分支机构、各住房公积金管理中心。——来源：`kb_324`

## 核心标准

### 1. 提取政策导向

通知确立了"租购并举"的提取政策方向，明确优先支持提取住房公积金支付房租，提取额度根据当地租金水平合理确定并及时调整。重点支持在缴存地或户籍地购买首套普通住房和第二套改善型住房，防止提取住房公积金用于炒房投机。——来源：`kb_326`

### 2. 离职提取规则

缴存职工与单位解除或终止劳动关系的，先办理个人账户封存。账户封存期间可选择以下方式处理：
- 在异地开立住房公积金账户并稳定缴存半年以上的，办理异地转移接续手续；
- 未在异地继续缴存的，封存满半年后可提取。

各地应于2018年6月底前完成当地政策的规范调整工作。——来源：`kb_326`

### 3. 审核时限标准

- 提取申请材料齐全、审核无误的，应即时办理；
- 需对申请材料进一步核查的，应在受理提取申请之日起3个工作日内办结。——来源：`kb_327`

### 4. 失信惩戒标准

- 违规提取的，记载失信记录并随个人账户转移；
- 已提取资金的，责令限期全额退回，在一定期限内限制提取和贷款；
- 逾期仍不退回的，列为严重失信行为，实施联合惩戒；
- 机关、事业单位及国有企业缴存职工违规情节严重的，向其所在单位通报。——来源：`kb_328`

### 5. 内部风险管理标准

住房公积金管理中心应合理设置提取受理、审核岗位，定期轮岗，形成"前台受理审核、后台重点复核、内审部门稽核"三道防线的内审稽核机制。委托商业银行受理审核的，将防控违规提取纳入业务考核，与受托资格和手续费挂钩。——来源：`kb_329`

### 6. 信息共享核查标准

建立住房公积金管理中心与公安、住建、人社、民政、不动产登记等部门及人民银行分支机构的信息共享机制，联网核查个人身份、户籍、房产交易、就业、社保、婚姻登记、不动产登记等信息。——来源：`kb_330`

## 适用条件

### 正常提取适用条件

通知对以下正常提取情形实行简化审核：
- 在缴存地租赁或购买自住住房；
- 偿还自住住房贷款本息；
- 离休退休；
- 与单位解除或终止劳动关系（需先办理账户封存）。——来源：`kb_326`, `kb_327`

### 严格审核情形

对以下情形，需严格审核住房消费行为和证明材料的真实性：
- 同一人多次变更婚姻关系购房；
- 多人频繁买卖同一套住房；
- 异地购房，尤其是非户籍地非缴存地购房；
- 非配偶或非直系亲属共同购房。——来源：`kb_327`

### 违规情形认定

通知明确以下行为构成违规：
- 伪造证明材料（如购房合同、发票、不动产权证书、结婚证等）；
- 虚构住房消费行为；
- 中介机构及其他组织参与违规提取；
- 涉嫌伪造及使用虚假证明材料的组织和个人。——来源：`kb_325`, `kb_332`

### 内部违规情形

对管理中心内部人员：
- 玩忽职守、失职渎职的，给予党纪政纪处分；
- 为违规提取提供便利、涉及职务犯罪的，移交司法机关处理。——来源：`kb_329`

## 办理流程

### 1. 正常提取审核流程

简化审核流程，积极推行网上审核和业务办理。材料齐全的即时办理；需进一步核查的3个工作日内办结。——来源：`kb_327`

### 2. 离职提取流程

解除/终止劳动关系 → 个人账户封存 → （可选路径①：异地开立账户并稳定缴存半年以上 → 办理异地转移接续）/（可选路径②：封存满半年 → 提取）。——来源：`kb_326`

### 3. 失信惩戒流程

违规提取 → 记载失信记录 → 已提取资金的责令限期全额退回 → 限制提取和贷款 → 逾期不退回的列为严重失信 → 向信用管理部门报送失信信息 → 联合惩戒。机关事业单位及国企职工情节严重的，同时向所在单位通报。——来源：`kb_328`

### 4. 跨地协查流程

提取审核需核查异地信息 → 提取审核地公积金管理中心商请信息产生地公积金管理中心 → 信息产生地公积金管理中心向当地相关部门代为核查 → 反馈核查结果。——来源：`kb_331`

### 5. 集中治理流程

公积金中心协调公安、通信、城管、网信等部门 → 全面清理违规提取信息 → 关停违规网站和违法电话 → 查处中介机构和其他组织 → 涉嫌伪造虚假证明材料的，移交公安等部门依法惩治。——来源：`kb_332`

### 6. 宣传引导流程

通过报纸、电视、网络等向社会公开办理要件、流程及时限 → 向缴存职工告知法律风险和责任 → 深入单位、社区宣传政策 → 公开曝光违规案例。——来源：`kb_333`
