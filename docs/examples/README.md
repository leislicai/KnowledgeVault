# 管线示例输出

本目录包含一次真实管线执行的产物——天水市住房公积金政策文档经过四阶段管线处理的结构化输出。

**执行时间：** 2026-06-12
**源文件：** 31 份政策文件（HTML/DOCX/PDF/DOC），覆盖天水市住房公积金归集、提取、贷款等业务
**领域配置：** gov-services（政务服务政策文件）
**总产出：** 338 个知识块 → 669 个实体 → 125 个 Wiki 页面 → 496 个 QA 问答对

